# PAU Revision Dojo Database

Base inicial de 500 ejercicios originales tipo PAU para cargar en una web/app.

## Campos
- id: identificador único.
- subject: asignatura.
- topic: tema.
- difficulty: baja, media o alta.
- exam_type: PAU / Selectividad.
- autonomous_community: comunidad autónoma o General España.
- year_reference: estilo de referencia.
- question: enunciado del ejercicio.
- solution: solución o guía de corrección.
- marks: puntuación orientativa.
- source_type: tipo de origen.
- tags: etiquetas para búsqueda/filtros.
